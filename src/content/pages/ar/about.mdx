## من أنا

أنا مهندس برمجيات من الجزائر، متخصص في تطوير الويب والتطبيقات المحمولة. أعمل مع React و Next.js و React Native و Expo على الواجهة الأمامية، و Node.js و Python و FastAPI على الخلفية.

تشمل خبرتي بناء أنظمة ERP، وتطبيقات مدعومة بالذكاء الاصطناعي، ومنصات SaaS، وسير عمل DevOps. أنا شغوف بإنشاء برامج عالية الأداء وسهلة الوصول ومُصممة بشكل جيد.

## ماذا أفعل

- **الواجهة الأمامية**: React, Next.js, TypeScript, Tailwind CSS, Shadcn UI
- **المحمول**: React Native, Expo
- **الخلفية**: Node.js, Express, Python, FastAPI
- **قاعدة البيانات**: PostgreSQL, MongoDB, Prisma, Drizzle
- **DevOps**: Docker, CI/CD, Vercel, Linux
