A full-featured e-commerce platform built with Next.js and TypeScript. Features include product catalog browsing, shopping cart management, secure JWT-based authentication, and a responsive UI powered by Shadcn components.

The project demonstrates end-to-end full-stack development with modern tooling, from server-side rendering and API routes to client-side state management and payment integration.
