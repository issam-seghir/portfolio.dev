## Project Overview

**Astralyn | Your Comprehensive Dashboard Experience** is a multi-module admin dashboard focused on analytics, operations, and productivity workflows. The product combines e-commerce-style data management with visualization and utility apps inside a single, customizable interface.

## Product Features

### Main features

- **🚀 Explore 6 fully equipped apps**: Kanban board, scheduler for appointments, markdown editor, diagram drawer, expense tracker, and map visualization
- **🔥 3 dynamic data grids**: Pagination, add/edit/delete (CRUD), filtering, searching, and more
- **🧪 Interactive data visualization**
- **⚡ Exporting capabilities**: Export charts, notes, and table data to **PDF**, **Excel**, and **CSV**
- **💻 Responsive** for mobile and desktop
- **💼 Everything cached** in **local storage**

### User customization options

- **🌓 Dark/light mode** (system by default)
- **🎨 3 themes**
- **🌍 Multiple languages** (English & Arabic)
- **✨ Extra effects** (experimental; e.g. rainbow blur background)

## Tech Stack

- **React + Vite** — Component architecture with a fast local development loop and efficient bundling
- **Joy UI (MUI Joy)** — Consistent design system, theming, and reusable UI primitives
- **Syncfusion** — Enterprise-style grids, charting, scheduler, and export tooling
- **React Router** — Multi-module dashboard navigation and route-based app composition
- **Leaflet** — Interactive map rendering for location-based views
- **SASS** — Custom styling layer for visual polish and fine-grained component control

## Implementation Highlights

- Built as a **modular dashboard shell** where each feature app (Kanban, scheduler, editor, etc.) behaves like a first-class module
- Used **reusable data-table patterns** (CRUD + filter + pagination) across multiple sections for consistency
- Integrated **export flows** directly into data and content screens to support practical admin use cases
- Added **locale-aware UI** for English and Arabic plus user preference persistence through local storage
- Optimized for **cross-device usage** while preserving information density on larger screens
