## The Problem

Creative Book needed a website that could match the energy and creativity of their educational content. Standard templates felt lifeless. They wanted something that would captivate visitors the moment they land, showcase their book catalog in an engaging way, and serve both Arabic and English-speaking audiences seamlessly.

## The Solution

I built a fully custom landing page with complex, interactive animations powered by Framer Motion, a modern bento grid layout, and full bilingual support (Arabic and English) using next-intl. The site is designed to feel like a curated experience, not just a page.

### Key Features

- **Interactive Complex Animations** with scroll-triggered, hover-responsive, and transition-based effects that bring every section to life
- **Bento Grid Layout** with modern, magazine-style content blocks following the latest UI trends
- **Bilingual Support** with full Arabic (RTL) and English (LTR) and seamless language switching
- **Newsletter Integration** powered by Mailchimp for reader engagement
- **Theme Toggler** with light and dark mode and smooth transitions
- **Optimized SEO and Performance** with meta tags, Open Graph, sitemaps, and Next/Image optimization

## Technical Decisions

Framer Motion was chosen for the animation layer because of its declarative API and deep React integration. It handles layout animations, gesture-based interactions, and scroll-linked effects with minimal overhead. The bento grid layout was implemented with CSS Grid and Tailwind, giving full control over responsive breakpoints.

next-intl handles internationalization with RTL layout support baked into the Tailwind config. Zustand manages client-side UI state with a minimal footprint. The entire project scores 95+ on Lighthouse across all metrics.

## What I Learned

This project was a masterclass in animation choreography. Learning to time and sequence dozens of independent animations so they feel cohesive rather than chaotic. It also deepened my understanding of RTL layout challenges and how to build truly bilingual experiences that feel native in both directions.
