A full-stack Discord-inspired application featuring real-time messaging, voice and video calls powered by LiveKit, server/channel management, and member roles. Built with Socket.io for instant bi-directional communication and Prisma for database management.

Supports direct messaging, file attachments, infinite message loading with scroll, and a responsive design that works across devices.
